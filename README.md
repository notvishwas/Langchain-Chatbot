# Basic LangChain Chatbot

created two chatbot using LangChain framework, one uses gemini-2.5-flash API, other uses local AI i create on my system using Ollama

## Installation

Clone the repo and run this command to install dependencies

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r require.txt
```


## Note

make sure to put API key / local ai setup in code
